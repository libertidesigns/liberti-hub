# LIBERTI Hub — Instalace jako aplikace

## Soubory
- `index.html` — hlavní aplikace
- `manifest.json` — PWA manifest
- `sw.js` — service worker (offline podpora)
- `icon-192.png` / `icon-512.png` — ikony aplikace

## Nasazení na GitHub Pages (zdarma, 5 minut)

1. Jdi na **github.com** → přihlas se (nebo vytvoř účet)
2. Klikni **+ New repository**
3. Název: `liberti-hub` — nastav jako **Private** (soukromé)
4. Klikni **Create repository**
5. Nahraj všechny soubory z této složky (drag & drop)
6. Jdi do **Settings → Pages**
7. Source: **Deploy from branch → main → / (root)**
8. Klikni **Save**

Za pár minut bude appka dostupná na:
`https://libertidesigns.github.io/liberti-hub/`

## Instalace jako aplikace

### Na počítači (Chrome)
1. Otevři výše uvedenou URL
2. V adresním řádku klikni na ikonu **⊞** (Nainstalovat)
3. Klikni Nainstalovat → LIBERTI Hub se otevře jako samostatná aplikace

### Na mobilu (Android)
1. Otevři URL v Chrome
2. Menu (⋮) → **Přidat na plochu**

### Na mobilu (iPhone/iPad)
1. Otevři URL v Safari
2. Sdílet (□↑) → **Přidat na plochu**

## Výsledek
- Samostatné okno bez záložek a adresního řádku
- Ikona na ploše / home screenu
- Funguje i offline (po prvním načtení)
- Automatický fullscreen na první interakci
